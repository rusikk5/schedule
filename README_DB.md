# Тестовая база данных для расписания

Добавлен минимальный сервер на Python со встроенной SQLite-базой.

## Что уже сделано

- Создается база `schedule.db`
- Добавляется группа: `ИВТ-б-о-241`
- Для группы добавлены:
  - `week1` (первая неделя)
  - `week2` (вторая неделя)
- Заполнены базовые школьные предметы, чтобы сразу увидеть пример
- В `Rozklad.html` включена автозагрузка данных из API
- В `admin.html` редактирование и добавление групп работают через API и сохраняются в SQLite

## Как запустить

1. Перейти в папку проекта.
2. Инициализировать базу:

```powershell
python app.py --init
```

3. Запустить API:

```powershell
python app.py
```

4. Открыть `Rozklad.html` для просмотра или `admin.html` для редактирования.

Если API запущен, страница подтянет данные из SQLite (`http://127.0.0.1:8000/api/schedule/all`).

## API

- `GET /api/health` - проверка API
- `GET /api/groups` - список групп
- `GET /api/schedule?group=ИВТ-б-о-241&week=week1` - расписание группы за неделю
- `GET /api/schedule/all` - все расписание в формате фронта
- `POST /api/groups` - добавить группу
- `PUT /api/schedule/cell` - обновить одну ячейку расписания

Пример `PUT /api/schedule/cell`:

```json
{
  "group": "ИВТ-б-о-241",
  "week": "week1",
  "row": 0,
  "col": 0,
  "value": "Лекция: Математика (Иванова Н.С.), ауд. 201"
}
```
